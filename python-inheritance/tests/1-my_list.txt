Test du module 1-my_list
-------------------------

Importation de la classe MyList:
    >>> MyList = __import__('1-my_list').MyList

Test 1: Liste normale avec des entiers positifs
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Test 2: Liste vide
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []

Test 3: Liste avec un seul élément
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.print_sorted()
    [1]

Test 4: Liste avec des nombres négatifs
    >>> my_list = MyList()
    >>> my_list.append(-5)
    >>> my_list.append(3)
    >>> my_list.append(-1)
    >>> my_list.append(0)
    >>> my_list.append(2)
    >>> print(my_list)
    [-5, 3, -1, 0, 2]
    >>> my_list.print_sorted()
    [-5, -1, 0, 2, 3]
    >>> print(my_list)
    [-5, 3, -1, 0, 2]

Test 5: Liste déjà triée
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(4)
    >>> my_list.print_sorted()
    [1, 2, 3, 4]

Test 6: Liste triée en ordre décroissant
    >>> my_list = MyList()
    >>> my_list.append(5)
    >>> my_list.append(4)
    >>> my_list.append(3)
    >>> my_list.append(2)
    >>> my_list.append(1)
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Test 7: Liste avec des doublons
    >>> my_list = MyList()
    >>> my_list.append(3)
    >>> my_list.append(1)
    >>> my_list.append(3)
    >>> my_list.append(2)
    >>> my_list.append(1)
    >>> my_list.print_sorted()
    [1, 1, 2, 3, 3]

Test 8: Initialisation avec une liste
    >>> my_list = MyList([5, 2, 8, 1, 9])
    >>> my_list.print_sorted()
    [1, 2, 5, 8, 9]
    >>> print(my_list)
    [5, 2, 8, 1, 9]

Test 9: MyList est une instance de list
    >>> my_list = MyList()
    >>> isinstance(my_list, list)
    True

Test 10: MyList a la méthode print_sorted
    >>> my_list = MyList()
    >>> hasattr(my_list, 'print_sorted')
    True

Test 11: Vérifier que print_sorted est appelable
    >>> my_list = MyList()
    >>> callable(my_list.print_sorted)
    True

Test 12: Liste avec de grands nombres
    >>> my_list = MyList()
    >>> my_list.append(1000000)
    >>> my_list.append(-1000000)
    >>> my_list.append(0)
    >>> my_list.print_sorted()
    [-1000000, 0, 1000000]

Test 13: Utilisation des méthodes héritées de list
    >>> my_list = MyList([3, 1, 2])
    >>> my_list.append(4)
    >>> my_list.extend([5, 0])
    >>> print(my_list)
    [3, 1, 2, 4, 5, 0]
    >>> my_list.print_sorted()
    [0, 1, 2, 3, 4, 5]

Test 14: Test avec tous des zéros
    >>> my_list = MyList([0, 0, 0, 0])
    >>> my_list.print_sorted()
    [0, 0, 0, 0]

Test 15: Liste avec des nombres très proches
    >>> my_list = MyList([1, 2, 1, 2, 1])
    >>> my_list.print_sorted()
    [1, 1, 1, 2, 2]
